# 角色

# 角色制作

新版本的角色制作工作量将大幅简化，你只需要按照以下步骤即可制作一般的角色：

# 创建角色id

在预设类型角色id中，添加一个值，用作该角色的id。

位置

玩家>玩家角色>角色id

# 添加一个玩家角色

在动作初始化玩家角色中，使用动作添加一个玩家角色来添加角色。

位置

玩家>玩家角色>初始化玩家角色

动作添加一个玩家角色有以下参数：

  • 角色id：使用前面创建的角色id。
  • 角色图片。
  • 角色技能图片。
  • 角色名称。
  • 角色介绍：目前没有效果，可以选择不填。
  • 角色技能名称。
  • 角色技能简介。
  • 角色技能详细介绍：改部分内容会作为“角色技能按钮”的提示。
  • 选择语音：目前没有效果。
  • 种族：可填写常量_Terran、_Protoss、_Zerg、_Void。没有实际意义，仅作为“酒馆百科”区分角色用。

角色数量上限

注意添加的角色总数量不能超过常量_玩家角色数量上限，否则请修改该常量。

# 设计角色技能

现在推荐使用“角色技能2.0”制作角色技能，因此本手册只讲解通过“角色技能2.0”制作角色技能。
在文件夹“角色技能2.0”中创建一个新文件夹，作为新建角色的专有文件夹。

位置

玩家>玩家角色>角色技能2.0

参考示例，创建相应的触发器，选择事件。在触发器中使用函数角色事件触发玩家和角色事件触发卡牌索引找到关联的玩家和卡牌索引。

角色技能按钮

你可以通过动作启用/禁用角色技能按钮，来开启关闭角色技能按钮。关闭时，事件角色使用技能将失效。
